# Matiboux.me

Yet Another Portfolio Website, but it's for Mati.


## Development

Run the following command to build & start the project for development:

```bash
docker compose \
  -f docker-compose.yml -f docker-compose.override.yml \
  --env-file .env \
  watch
```
